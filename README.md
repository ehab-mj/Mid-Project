# 🎉 Event Tune

Event Tune is a modern **all-in-one event booking website** that makes planning an event **faster, easier, and way more organized**.

Instead of searching on Instagram/Facebook, messaging multiple vendors, and comparing prices manually — **Event Tune brings everything into one place** so users can build their event package in minutes.

---

## 🚀 What is Event Tune?

**Event Tune** helps users book everything they need for an event from a single platform:

- 🎧 DJ
- 🎨 Decoration
- 📸 Photographer
- 🏢 Venue

Users can plan any type of event, such as:

- 💍 Wedding
- 🎂 Birthday Party
- 🏢 Corporate Event
- 🎉 Anniversary
- 🎓 Graduation
- ➕ Other

The user selects what they need, reviews the full package, and sends a booking request.  
Providers (like DJs) receive the request and can **Accept ✅ or Deny ❌** it.

## 👤 User Roles

### 1) Regular User (Client)
The person planning the event.

**Regular user can:**
- Create an event
- Select event type + event details (date, guests, etc.)
- Book DJ / Decoration / Photographer / Venue
- Review full package before sending
- Send booking request to providers
- Track request status (Pending / Accepted / Rejected)

---

### 2) DJ User (Provider)
The person/service who receives event requests.

**DJ user can:**
- Receive booking requests
- View full event package details
- Accept ✅ or Deny ❌ requests
- Manage confirmed bookings

---

## 🛠 How It Works (Step-by-Step)

### Step 1 — Create Event
User chooses the event type and fills details like:
- date & time
- location
- number of guests

### Step 2 — Select Services
User browses categories and selects what they need:
- DJ
- Decoration
- Photographer
- Venue

### Step 3 — Review Package
A summary shows everything selected:
- event details
- chosen services
- total price estimate

### Step 4 — Send Request
The full package is sent to providers.

### Step 5 — Provider Response
Providers review and decide:
- ✅ Accept
- ❌ Deny

User gets the final status.
